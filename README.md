# Análisis de precios de competencia mediante web scraping

Este proyecto consiste en la recolección, limpieza y análisis de datos de precios de productos en tiendas en línea, con el objetivo de apoyar la definición de estrategias de precios competitivos para una tienda de ropa materna.

## Objetivo

Analizar los precios de la competencia mediante web scraping y procesamiento de datos para identificar oportunidades de mejora en la estrategia comercial.

## Tecnologías utilizadas

- Python
- Pandas
- Google Colab
- Machine Learning (Regresión lineal, regresión logística, Random Forest)

## Proceso del proyecto

1. Extracción de datos desde páginas web de la competencia mediante web scraping  
2. Limpieza y transformación del dataset  
3. Análisis exploratorio de datos  
4. Aplicación de modelos de machine learning  
5. Interpretación de resultados para toma de decisiones  

## Contexto

Proyecto académico desarrollado en colaboración con una compañera, aplicado a un caso real de negocio: Mundo Materno Bogotá.
